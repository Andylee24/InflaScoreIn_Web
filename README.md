# InflaScore Web

Static marketing and legal pages for InflaScore. Built for Cloudflare Pages with no server required.

## Structure

- `public/index.html` - landing page prototype
- `public/privacy.html` - draft Privacy Policy for App Store preparation
- `public/terms.html` - draft Terms & Conditions
- `stitch-reference/` - downloaded Stitch reference HTML and screenshot

## Cloudflare Pages

Use these settings:

- Framework preset: None
- Build command: leave empty
- Build output directory: `public`

The legal text is a product draft, not legal advice. Review before publishing.
